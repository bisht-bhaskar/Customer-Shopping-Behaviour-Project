# Customer-Shopping-Behavior-Project
End to end Data Analysis Project of Customer Shopping Behavior using Python, Sql and Power BI.

📊 Customer Shopping Behavior Analysis Dashboard

📌 Project Overview
This project focuses on analyzing customer shopping behavior to uncover meaningful insights that can help businesses make data-driven decisions. The dashboard provides a comprehensive view of sales trends, customer preferences, and product performance.
Using data visualization and analytical techniques, this project highlights key patterns in purchasing behavior and identifies opportunities to improve profitability and inventory planning.

🚀 Features
📈 Sales Performance Analysis – Track overall sales trends and revenue patterns
🛍️ Customer Behavior Insights – Understand purchasing habits and preferences
🏷️ Category-wise Analysis – Identify top-performing product categories
💸 Discount Impact Analysis – Evaluate how discounts affect sales and profit
📦 Inventory Insights – Analyze product distribution and weight across categories
🔍 Dynamic Dashboard – Interactive filters for better exploration of data

🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
SQL – Data extraction and querying
Python (Pandas, NumPy) – Data cleaning and transformation
Excel – Initial data handling and preprocessing

📊 Key Insights
Identified top 5 categories offering the highest discount percentages
Grouped products into categories like Low, Medium, and Bulk based on quantity
Calculated total inventory weight per category
Discovered patterns in customer purchasing behavior
Highlighted high-profit and low-performing segments
